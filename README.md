# 自动滚屏工具

一个 Windows 桌面小工具，可以为任意窗口开启自动滚屏。

## 功能

1. **指定目标窗口**：选择需要自动滚屏的程序窗口。
2. **调节滚动速度**：1 ~ 100 档线性均匀可调，第 1 档为原 1 档的 90%，第 100 档为原 50 档。按键盘 **← / →** 可快速减/加 1 档。
3. **选择滚动方向**：支持向上、向下、向左、向右，默认向下；滚屏过程中可随时热切换方向与速度。
4. **全局快捷键**：默认 `Ctrl + Alt + S`，随时开启 / 关闭自动滚屏。
5. **兼容模式**：普通模式不抢夺焦点，适合浏览器、PDF 阅读器等现代程序；窗口缩放/最小化后若无法滚动，或某些旧程序不响应，可切换到兼容模式。兼容模式需要把鼠标移到目标窗口上方（或让目标窗口为当前激活窗口）才生效。
6. **系统托盘**：点击关闭按钮会隐藏到托盘；右键托盘图标可选择【显示主窗口】或【退出程序】。
7. **配置持久化**：目标窗口、速度、方向、快捷键、兼容模式均自动保存。

## 使用方法

1. 下载 `release-portable/auto-scroll.exe`（或解压 zip）。
2. 直接双击运行，无需安装。
3. 在界面中：
   - 点击「刷新」选择要滚屏的目标窗口；
   - 拖动滑块设置速度；
   - 点击「修改」录制新的全局快捷键；
   - 点击「开始滚屏」或按快捷键开始自动滚屏。
4. 点击关闭按钮会隐藏到系统托盘；右键托盘图标可选择【显示主窗口】或【退出程序】。

> 系统要求：Windows 10 / 11（已内置或已安装 WebView2 Runtime）。
